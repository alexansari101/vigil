# Code Agent Skill

Complete the requested task or the next available task on the task list if one was not specifically provided.

[] Follow the instructions in `task_list.md` to complete the task.
[] Re-review the instructions at the top of task_list.md to double check that all steps were completed.
[] The final step after a thorough review and testing is to commit and merge your changes.
